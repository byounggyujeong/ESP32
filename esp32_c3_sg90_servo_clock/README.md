🕐 SG90 서보모터 7세그먼트 시계

ESP32-C3 + PCA9685 두 대로 28개의 SG90 서보모터를 제어하여 시:분을 7세그먼트 형태로 표시하는 WiFi 인터넷 시계입니다.


📦 구성품

부품수량비고ESP32-C3 Super Mini1개WiFi + 메인 컨트롤러PCA9685 (16채널 PWM 드라이버)2개I2C 주소 0x40, 0x42SG90 서보모터28개7세그먼트 × 4자리6V 3A SMPS1개서보 전원 공급점퍼선 / 브레드보드-배선용

⚠️ 주의: SMPS와 ESP32의 GND는 반드시 공통으로 연결해야 합니다. 공통 그라운드가 없으면 I2C 통신 오류 및 서보 오동작이 발생할 수 있습니다.




⚙️ 동작 방식


부팅 시 NVS(Preferences)에 저장된 WiFi 정보로 자동 접속
접속 실패 시 시리얼 모니터(SSID,PASSWORD 형식)로 입력 대기 → 저장 후 재부팅
NTP 서버(KST, UTC+9)로 시간 동기화
1초마다 시간 확인 후 변경된 자리만 순차적으로 서보 구동 (분1→분10→시1→시10, 각 400ms 간격)
그룹별로 동작 완료 400ms 후 개별 PWM 전력 차단 (발열/소비전력 절감)
설정된 수면 시간(기본 22시~7시) 진입 시 즉시 전체 전력 차단



🌐 웹 설정 페이지

ESP32 IP로 접속하면 다음 기능을 사용할 수 있습니다.

기능설명12/24시간 모드체크박스로 전환수면 시간 설정시작/종료 시(時) 지정, 해당 시간엔 서보 동작 중지서보 오프셋 보정28개 서보 개별 펄스 오프셋(-100~+100, 10단위 조절)🔧 88:88 테스트 모드전체 서보를 8 모양으로 고정, 오프셋 실시간 보정용🔢 숫자별 서보 테스트각 자리 숫자를 0~9 중 직접 선택해서 개별 확인


🔧 서보 캘리브레이션 값 (실측 기준)

PWM 펄스 허용 범위 : 100 ~ 600
중앙(ON) 위치      : 375
OFF 위치           : 150 또는 550 (세그먼트 방향에 따라 다름)

digits[10][7] 배열은 위 실측값을 기반으로 0~9 숫자별 7세그먼트 ON/OFF 패턴을 구성했습니다. 실제 조립 후 개체차가 있을 수 있으므로 88:88 테스트 모드 → 오프셋 보정 순서로 미세조정하는 것을 권장합니다.
