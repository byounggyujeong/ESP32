# 🕐 SG90 서보모터 7세그먼트 시계

> **최신 버전: v2.0** (`firmware/servo_clock_wifi_set/`) · 변경 내역은 [CHANGELOG.md](CHANGELOG.md) 참고
> 이전 버전(v1.0)은 `archive/servo_clock_final/`에 보관 중입니다.

ESP32-C3 + PCA9685 두 대로 28개의 SG90 서보모터를 제어하여 시:분을 7세그먼트 형태로 표시하는 WiFi 인터넷 시계입니다.

---

## 📦 구성품

| 부품 | 수량 | 비고 |
|------|------|------|
| ESP32-C3 Super Mini | 1개 | WiFi + 메인 컨트롤러 |
| PCA9685 (16채널 PWM 드라이버) | 2개 | I2C 주소 0x40, 0x42 |
| SG90 서보모터 | 28개 | 7세그먼트 × 4자리 |
| 6V 3A SMPS | 1개 | 서보 전원 공급 |
| 점퍼선 / 브레드보드 | - | 배선용 |

---

## 🔌 회로 연결도

```
                    ┌──────────────────────────┐
                    │   ESP32-C3 Super Mini    │
                    │  GPIO8=SDA  GPIO9=SCL    │
                    └────────────┬─────────────┘
                                 │ I2C / 3.3V
                  ┌──────────────┴──────────────┐
                  │                             │
                  ▼                             ▼
     ┌────────────────────────┐    ┌────────────────────────┐
     │   PCA9685 #1 (pwm1)    │    │   PCA9685 #2 (pwm2)    │
     │   I2C 0x40 · 분 표시    │    │   I2C 0x42 · 시 표시    │
     └────────────┬────────────┘    └────────────┬────────────┘
                  │ PWM                          │ PWM
                  ▼                              ▼
     ┌────────────────────────┐    ┌────────────────────────┐
     │      분 서보 14개        │    │      시 서보 14개        │
     │   채널 0~6, 9~15         │    │   채널 0~6, 9~15         │
     └────────────▲────────────┘    └────────────▲────────────┘
                  │                              │
                  └──────────────┬───────────────┘
                                 │ V+ (서보 전원)
                    ┌────────────┴─────────────┐
                    │       6V 3A SMPS         │
                    │  GND는 ESP32와 공통 연결   │
                    └──────────────────────────┘
```

### ESP32-C3 핀맵 참고

| ESP32-C3 핀 | 연결 대상 | 설명 |
|---|---|---|
| GPIO8 | PCA9685 SDA (x2 공통) | I2C 데이터 |
| GPIO9 | PCA9685 SCL (x2 공통) | I2C 클럭 |
| 3.3V | PCA9685 VCC (로직, x2) | 로직 전원 |
| GND | PCA9685 GND + SMPS GND | **공통 그라운드 필수** |

### PCA9685 채널 배치

| 보드 | 채널 0~6 | 채널 7~8 | 채널 9~15 |
|------|----------|----------|-----------|
| pwm1 (0x40) | 분 1자리 (S0~S6) | 미사용 | 분 10자리 (S0~S6) |
| pwm2 (0x42) | 시 1자리 (S0~S6) | 미사용 | 시 10자리 (S0~S6) |

### 세그먼트 위치 정의

```
 ─── S0 (상단 가로)
|   |
S5  S1 (좌상, 우상 세로)
 ─── S2 (중단 가로)
|   |
S6  S3 (좌하, 우하 세로)
 ─── S4 (하단 가로)
```

> ⚠️ **주의**: SMPS와 ESP32의 GND는 반드시 공통으로 연결해야 합니다. 공통 그라운드가 없으면 I2C 통신 오류 및 서보 오동작이 발생할 수 있습니다.

## ⚙️ 동작 방식

1. 부팅 시 NVS(Preferences)에 저장된 WiFi 정보로 자동 접속
2. 접속 실패 시 **`ServoClock-Setup`** 오픈 AP가 켜짐 → 핸드폰으로 접속하면 캡티브 포털이 자동으로 뜸 → 주변 WiFi 선택 + 비밀번호 입력 → 저장 즉시 연결 시도 후 **포털 화면에 연결된 IP 주소 표시**
3. NTP 서버(KST, UTC+9)로 시간 동기화
4. 1초마다 시간 확인 후 변경된 자리만 순차적으로 서보 구동 (분1→분10→시1→시10, 각 400ms 간격, 자리 내 세그먼트도 한 개씩 순차 이동)
5. 그룹별로 동작 완료 400ms 후 개별 PWM 전력 차단 (발열/소비전력 절감)
6. 설정된 수면 시간(기본 22시~7시) 진입 시 즉시 전체 전력 차단

### 시계 접속 주소 확인 방법 (3가지)

| 방법 | 설명 |
|------|------|
| 시리얼 모니터 | 부팅 로그에 IP 그대로 출력 |
| WiFi 설정 포털 | 핸드폰으로 설정 직후 화면에 IP 바로 표시 |
| mDNS | `http://servoclock.local` 로 IP 몰라도 접속 가능 |

## 🌐 웹 설정 페이지

ESP32 IP(또는 `servoclock.local`)로 접속하면 다음 기능을 사용할 수 있습니다.

| 기능 | 설명 |
|------|------|
| 12/24시간 모드 | 체크박스로 전환 |
| 수면 시간 설정 | 시작/종료 시(時) 지정, 해당 시간엔 서보 동작 중지 |
| 서보 오프셋 보정 | 28개 서보 개별 펄스 오프셋(-100~+100, 10단위 조절) |
| 🔧 88:88 테스트 모드 | 전체 서보를 8 모양으로 고정, 오프셋 실시간 보정용 |
| 🔢 숫자별 서보 테스트 | 각 자리 숫자를 0~9 중 직접 선택해서 개별 확인 |
| 📶 WiFi 재설정 | 저장된 WiFi 정보 삭제 + 재부팅 → `ServoClock-Setup` 포털 재진입 |

## 🔧 서보 캘리브레이션 값 (실측 기준)

```
PWM 펄스 허용 범위 : 100 ~ 600
중앙(ON) 위치      : 375
OFF 위치           : 150 또는 550 (세그먼트 방향에 따라 다름)
```

`digits[10][7]` 배열은 위 실측값을 기반으로 0~9 숫자별 7세그먼트 ON/OFF 패턴을 구성했습니다. 실제 조립 후 개체차가 있을 수 있으므로 **88:88 테스트 모드 → 오프셋 보정** 순서로 미세조정하는 것을 권장합니다.

## 📝 라이브러리 의존성

아두이노 IDE → 스케치 → 라이브러리 포함하기 → 라이브러리 관리에서 검색 후 설치

- `Adafruit PWM Servo Driver Library`
- ESP32 보드 패키지 (보드 매니저 URL: `https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json`)

**보드 설정**: 툴 → 보드 → ESP32 Arduino → `ESP32C3 Dev Module`

> v2.0부터 `WiFiSetup.h` / `WiFiSetup.cpp`가 함께 필요합니다. `.ino` 파일과 같은 폴더에 두면 아두이노 IDE에서 자동으로 탭으로 인식됩니다. 별도 라이브러리 설치는 필요 없습니다 (Preferences/WebServer/DNSServer/ESPmDNS 모두 ESP32 보드 패키지 내장).

## ⚠️ 트러블슈팅 노트

| 증상 | 원인 | 해결 |
|------|------|------|
| `Adafruit_PWMServoDriver.h` 없음 | 라이브러리 미설치 | 라이브러리 매니저에서 설치 |
| 컴파일 시 `ld.exe` 경로 에러 | 한글 사용자명 경로 문제 | 스케치북 위치를 영문 경로로 변경 |
| 업로드 시 `COM 포트 busy` | 시리얼 모니터 점유 중 | 시리얼 모니터/플로터 닫고 재시도 |
| I2C 스캔 시 0x40만 검출 | PCA9685 A0 점퍼 미납땜 | 코드 주소를 0x40으로 맞추거나 점퍼 납땜 |
| USB만 연결해도 동작 | 순차 구동이라 전류 여유 있음 | 장시간 사용 시엔 SMPS 필수 연결 권장 |
| 모터 전혀 무반응 | PCA9685 V+ 미연결 (SMPS 없음) | V+ 라인에 전원 공급 필수 |
| `redefinition of ... STEP_DELAY` | 상수 이름 중복 선언 | 새 딜레이 상수는 다른 이름(`SEGMENT_DELAY` 등) 사용 |
